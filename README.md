# Python Mini Applications

A collection of small Python scripts focused on logic building using conditions, loops, and functions.

## Applications

- **calculator.py**: Basic arithmetic operations (add, subtract, multiply, divide)  
- **string_utils.py**: String manipulation utilities like reversing strings and counting vowels  
- **number_operations.py**: Number-related functions including prime checking and factorial calculation  

## Usage

Run each script individually to see example outputs and understand basic programming concepts.

## Technologies

- Python 3.x
